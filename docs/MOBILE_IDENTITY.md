# Guía de Identidad Visual – Stock Mobile (Android)

Este documento define las bases visuales para la aplicación nativa Android, asegurando una transición fluida y una estética coherente con la versión web.

## Paleta de Colores (Material Design 3 Mapping)

| Rol | Color Hex | Uso en Mobile |
| :--- | :--- | :--- |
| **Primary** | `#0B1F33` (Navy) | Barra superior, botones principales y fondo de login. |
| **Secondary** | `#F2AD3D` (Amber) | Acentos, indicadores de estado y botones de confirmación. |
| **Tertiary** | `#06286F` (Navy Accent) | Variaciones de superficie y contenedores secundarios. |
| **Background** | `#041120` (Navy Deep) | Fondo principal de la aplicación (Dark Mode por defecto). |
| **Surface** | `rgba(255, 255, 255, 0.05)` | Tarjetas de productos y detalles de pedidos (Glassmorphism). |
| **Text Primary** | `#FFFFFF` | Títulos y texto de alto énfasis. |
| **Text Secondary**| `rgba(248, 245, 239, 0.75)`| Descripciones y etiquetas. |

## Tipografía (Google Fonts)

*   **Display / Títulos:** [Manrope](https://fonts.google.com/specimen/Manrope) (Bold/Medium).
*   **Cuerpo / Datos:** [Inter](https://fonts.google.com/specimen/Inter) (Regular/Medium).

## Principios de Diseño
1.  **Glassmorphism Moderado:** Usar superficies translúcidas sobre fondos profundos para mantener el aspecto premium.
2.  **Micro-interacciones:** Retroalimentación háptica y visual inmediata en escaneos y cambios de stock.
3.  **Jerarquía Visual:** El stock crítico y los avisos de "Bajo Stock" deben usar el color **Amber** para visibilidad inmediata.

## Iconografía
Utilizaremos **Material Symbols (Rounded)** para una apariencia amigable y moderna.
